# 更新日志

## [V0.0.4] - 2026-05-28

### 🐛 Bug 修复

#### 🌐 GitHub Pages 部署 404 修复
- **原因**：GitHub Pages 默认使用 Jekyll 处理站点文件，导致静态资源无法正确加载而返回 404
- **修复**：在 `public/` 目录添加 `.nojekyll` 文件，明确禁用 Jekyll 处理
- **效果**：`_config.yml` + `.nojekyll` 双重保障，确保纯静态 SPA 正常部署

## [V0.0.3] - 2026-05-28

### ✨ 新增功能

#### 🏷️ 标签筛选与排序
- **标签道藏**：标签云展示所有标签，点击筛选对应文档
- **文档列表**：展示文档标题、标签、阅读量，支持按阅读量升序/降序排序
- **标签道统**：标签分布统计可视化，条形图展示每标签的文档数和总阅读量

#### ⚙️ 灵蕴天碑 · Gist 数据管理
- **可视化数据面板**：阅读量排行表格、修仙数据卡片、其他数据分类展示
- **原始 JSON 查看/编辑**：预览完整 JSON，支持在线编辑并保存回 Gist
- **修改阅读量**：管理模式中可修改任意文档的阅读量
- **密码保护**：清空/删除/修改操作需 SHA-256 验证密码
- **管理模式**：连续点击标题 3 次解锁管理按钮

#### 🔗 统一 Gist 存储层
- **gistStore**：统一管理页面阅读量、修仙历程数据，自动降级 localStorage

### 🛡️ 安全
- 管理密码以 SHA-256 哈希硬编码，不存明文
- 破坏性操作（清空/删除/修改）均需密码确认
- 管理按钮默认隐藏，需三击标题解锁

## [V0.0.2] - 2026-05-27

### ✨ 新增功能

#### ⚡ 修仙成神系统
- **修仙引擎**：基于文档量、分类体系、标签数量计算的修为值系统
- **境界体系**：10 大境界（练气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫→飞升成神），每境范围翻倍
- **道纹徽章**：每个境界有专属道纹图形，纯 CSS 实现（灵纹→丹纹→星纹→破界纹→旭日金星）
- **修炼面板**：首页展示当前境界、修为值、小层级进度条、灵力速度四维指标
- **突破动画**：80 粒子飘散 + 双层光环扩散 + 光柱升起的炫酷升级动画
- **成神时间线**：所有突破记录可视化时间轴，点击任一节点可回放动画
- **修炼说明书**：右上角「📜 修炼指南」按钮，含境界总览、修为规则、道纹图鉴、突破动画说明
- **数据持久化**：历程数据同步至 GitHub Gist（跨设备），自动降级 localStorage
- **修炼指南文档**：`docs/修炼指南.md` 完整文档
